# श्री खेड़ापति सरकार मंदिर - Shri Khedapati Sarkar Mandir Website

A modern, spiritual, and culturally rich website for Shri Khedapati Sarkar Mandir in Madhya Pradesh, India.

## 🎯 Features

- **Spiritual Design**: Saffron, red, gold color scheme with traditional Indian aesthetics
- **Bilingual Support**: Hindi and English content
- **Responsive Design**: Mobile-first approach with modern UI/UX
- **Live Darshan**: Embedded streaming capability
- **Online Donations**: Secure payment gateway integration
- **Puja Booking**: Online service booking system
- **Event Calendar**: Festival and event information
- **Gallery**: Categorized photos and videos
- **PWA Support**: Progressive Web App functionality

## 🚀 Quick Start

1. **Clone or Download** the project files
2. **Open** `index.html` in a web browser
3. **Customize** content, images, and contact information
4. **Deploy** to your web hosting service

## 📁 Project Structure

```
khedapati-mandir-website/
├── index.html              # Main HTML file
├── styles/
│   └── main.css           # Main stylesheet
```

## 🔧 Customization

### Adding Images
1. Replace placeholder images in the `assets/` folder
2. Update image paths in `index.html`
3. Optimize images for web (compress, use WebP format)

### Updating Content
1. Edit text content in `index.html`
2. Modify styles in `styles/main.css`
3. Update functionality in `scripts/main.js`

### Contact Information
Update the following in `index.html`:
- Phone number
- Address
- Social media links
- WhatsApp number

### Payment Integration
To enable real payments:
1. Sign up with Razorpay/PayU
2. Replace demo payment code in `scripts/main.js`
3. Add your payment gateway credentials

## 🌐 Deployment

### Static Hosting
- **Netlify**: Drag and drop the folder

- **Firebase Hosting**: Use Firebase CLI

### Traditional Hosting
- Upload files via FTP/SFTP
- Ensure `index.html` is in the root directory
- Configure HTTPS for security

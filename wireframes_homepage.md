## 🎨 Design Elements

- **Color Scheme**: Saffron (#FF9933), Red (#CC0000), Gold (#FFD700), White (#FFFFFF)
- **Typography**: Devanagari script for headings, clean sans-serif for body text
- **Icons**: 🔔 Bells, 🕯️ Diyas, 🪔 Lotus, 🕉️ OM symbol, 🙏 Namaste
- **Animations**: Smooth scroll, fade-in effects, floating action buttons
- **Interactive Elements**: Hover effects, button animations, image carousels
